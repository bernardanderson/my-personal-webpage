"Eyes On The Net"

Eyes On The Net is a web application for securely interfacing with and displaying local and remote webcams which otherwise might have poor or no standalone security. Snapshots are handled server-side for later retrieval in a DVR-type fashion and camera locations can be graphically viewed via Google Maps.

It was written using .NET Core (in Linux), Angular 2 with TypeScript, a standalone MySQL server for Db storage, JSON Web Tokens for user authentication and styled using Semantic-Ui.

"Cheque-Yourself!"

Paper checks are still a "thing" of modern life. After trying to find software that would allow an individual to print their own checks (as a cost-saving measure), it was determined that available check-writing software was expensive, horribly out-dated and platform specific.

"Cheque-Yourself!" allows the printing of checks from any web browser as well as keep track of checking & saving finances.

"Gauntlet"

One of the more fun front-end group projects was "Gauntlet", a JavaScript exercise which had the student utilizing prototypical inheritance for property creation and modification. The user creates a player, selects a "class" and a weapon in order to engage in a 1-on-1 fight with an Orc.

After completing the JavaScript portion, our group had fun with the project by adding a 1980's 8-bit theme as well as CSS animations for the actual battle scene.
