"Cheque-Yourself!"

Paper checks are still a "thing" of modern life. After trying to find software that would allow an individual to print their own checks (as a cost-saving measure), it was determined that available check-writing software was expensive, horribly out-dated and platform specific.

"Cheque-Yourself!" allows the printing of checks from any web browser as well as keep track of checking & saving finances.

"Gauntlet"

One of the more fun Front-End group projects was "Gauntlet", a javascript excercise which exposed the students to multiple objects utilizing prototypical inheritence for property creation and modification. The user creates a player, selects a "class" and a weapon in order to engage in a 1-on-1 fight with an Orc.

After completing the javascript portion, our group had fun with the project by adding a 1980's 8-bit theme as well as CSS animations for the actual battle scene.
